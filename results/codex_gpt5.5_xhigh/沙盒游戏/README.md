# 落沙实验室

一个自包含的 HTML/CSS/JavaScript 落沙模拟游戏。直接打开 `index.html` 即可运行。

## 已实现

- 网格粒子物理：重力、密度置换、粉体堆积、液体流动、气体上升。
- 数据驱动材质：沙子、水、石头、木头、火、烟、油、岩浆、酸、冰、蒸汽、植物、盐、金属。
- 温度与反应：燃烧、热传导、结冰、蒸发、冷凝、岩浆遇水、酸蚀、植物吸水生长。
- 游戏交互：材质笔刷、半径调节、圆形/方形/喷洒笔刷、暂停、单步、清空、降温、场景预设。
- 响应式界面：桌面双栏工作台，移动端纵向布局。

## 文件

- `index.html`：页面结构。
- `styles.css`：界面布局、控件样式、响应式规则。
- `engine.js`：材质定义和物理模拟引擎。
- `app.js`：渲染循环、输入事件、场景预设和 UI 状态。
