# 落沙模拟游戏 - 规格说明书

## 1. 项目概述

- **项目名称**: Sand Fall Simulator
- **类型**: 物理模拟游戏
- **核心功能**: 基于Matter.js物理引擎的落沙模拟，支持多种材质，具有交互式沙漏效果
- **目标用户**: 休闲玩家、物理模拟爱好者

## 2. 视觉与渲染规格

### 场景设置
- **画布**: 全屏Canvas，响应式尺寸
- **背景**: 深色渐变 (#1a1a2e → #16213e)
- **边框**: 模拟沙漏的边框效果

### 材质视觉
| 材质 | 颜色 | 粒子大小 | 密度 | 摩擦力 |
|------|------|---------|------|--------|
| 普通沙 | #e6c86e | 4px | 0.8 | 0.1 |
| 粗沙 | #a67c52 | 6px | 1.2 | 0.3 |
| 盐 | #f5f5f5 | 3px | 0.5 | 0.05 |
| 糖 | #ffd93d | 5px | 0.7 | 0.08 |
| 活性炭 | #2d2d2d | 4px | 0.6 | 0.15 |
| 彩色沙子 | 彩虹渐变 | 4px | 0.8 | 0.1 |

### 视觉效果
- 粒子带有轻微的发光效果
- 材质切换有颜色过渡动画
- 落点有微小的碰撞粒子特效

## 3. 物理模拟规格

### 引擎
- **物理引擎**: Matter.js (CDN)
- **渲染**: Matter.Render + 自定义Canvas叠加

### 物理参数
- **重力**: { x: 0, y: 1 }
- **时间步长**: 1000/60
- **粒子数量上限**: 2000
- **碰撞检测**: 启用

### 材质物理属性
```
普通沙:   density=0.001, friction=0.1, restitution=0.05
粗沙:     density=0.002, friction=0.3, restitution=0.02
盐:       density=0.0008, friction=0.05, restitution=0.1
糖:       density=0.001, friction=0.08, restitution=0.08
活性炭:   density=0.0015, friction=0.15, restitution=0.03
```

## 4. 交互规格

### 控制方式
- **鼠标点击**: 在点击位置生成粒子
- **鼠标拖拽**: 连续生成粒子流
- **材质选择**: 底部UI按钮切换材质
- **清除按钮**: 清除所有粒子

### UI元素
- 顶部标题栏
- 底部材质选择栏（6种材质）
- 粒子计数器显示
- 清除按钮
